Log sources / indexes

How do you know your volume?

Pick one way to enter volume: Daily GB or EPS — not both at once. Both mean raw / pre-indexed ingest (license-style), not compressed on-disk size. Under each number the other unit is estimated from average event size.

Only the selected mode is editable. The other value appears in small text under each input (Daily↔EPS using event size). Sources without an EPS inherit the average EPS of sources that already have one.

EPS = raw events arriving from the source (before Splunk compression). With average event size it becomes Daily raw GB; on-disk ≈ that × Comp.
