### Using the calculator

1. **Start wizard** — Topology → Retention → Sources → Review → Calculate.  
2. Fill the fields you know (source volumes, optional total daily GB, optional disk budgets). There is no separate “mode” step.  
3. **Export URL** — copy a shareable link for docs or colleagues.  
4. **Import** — paste that link (or choose a `.json` plan file).  
5. Language: English / فارسی.

Hover dotted labels for formulas and official Splunk documentation links.
